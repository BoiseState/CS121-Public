The chap06 examples are arranged into multiple packages to organize them better. Here is what each package contains:

  • default package: Contains examples from the textbook.

  • basics package: Contains examples that demonstrate one component/container at a time.

  • converter1, converter2, converter3 packages: A series that illustrates GUI design elements.

  • orderapplication package: Contains a more complex layout example.

  • textedit package: Contains a notepad like text editor in Java. Uses menus extensively.

  • textedit.advanced package: Contains a more advanced version of the text editor example.

  • misc package: Contains miscellaneous examples.
